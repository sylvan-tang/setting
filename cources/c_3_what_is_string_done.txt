字符串的定义
与数字区分开来看，平时我们使用到的文字除了数字之外就都得用字符串的结构存储到计算机中
而且如果需要在文字中混入数字，那么数字也必须是文字中的一部分
那在 python 中就得有和数字区别的定义方式

在 python 中，有以下四种定义字符串的方式
1、用英文双引号 " 将文字括起来，比如 "abc" "我的前半生" "123。木头人"，这里有三个字符串
2、用英文单引号 ' 将文字括起来，比如 'abc' '' '1份早餐'，这里有几个字符串？
3、用前后三个英文双引号 """ 把字符串括起来，比如 """Don't worry, be happy."""，这里只有一个字符串
4、用前后三个英文单引号 ''' 把字符串括起来，比如 '''Some one like you.'''，这里也只有一个字符串
所以，说到这里可以发现：
1、单引号和双引号的使用是完全等价的，也就是说，"abc" 等于 'abc' """abc""" 也等于 '''abc'''
2、其实三个双引号或三个单引号的功能在某些情况下，和单个单双引号的功能也一致，如第一点的示例里面，四个字符串都是相等的，
   这是因为三个双引号和三个单引号的主要用处，是给多行字符串用的，比如有一篇文章要保存到计算机里，总得保留文章的段落，不能都放在同一行
3、既然单引号和双引号是用来定义字符串的，但是如果字符串里面有单引号和双引号怎么办呢？
   可以在有双引号的地方，用单引号定义字符串，比如 '我有一本书叫"致命的百合货"，是一本自传'
   同理可以有单引号的地方，用双引号定义字符串，比如 "I don't care"
4、所以说用哪种引号定义字符串，中间就不能出现哪种引号
